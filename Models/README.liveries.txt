Making liveries for the CRJ700 series:

1. Make your livery using any of the 3 provided paintkits

2. Save finished PNG to CRJ700-family/Models/Liveries/<insert variant here>/

3. Copy and modify appropriate XML file, example:

<?xml version="1.0" encoding="UTF-8" ?>

<PropertyList>

	<sim>
		<model>
			<livery>
				<name>United Express/Mesa Airlines</name>
				<texture>Liveries/CRJ700/ASH.png</texture>
				<registration>N502MJ</registration>
			</livery>
		</model>
	</sim>

</PropertyList>
